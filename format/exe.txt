Output format for the assembler and linker
Header total 32 bytes
4 Identification bytes which must be:
    0x04 0x4F 0x52 0x49
1 Type byte
    Possible values
        0 = INVALID
        1 = OBJECT
        2 = EXECUTABLE
        3 = KERNEL FILE
1 Endianess byte (0 = little 1 = big)
1 Arch byte
    Possible values
        0 = INVALID
        2 = x64
8 Entry bytes
8 Section offset bytes
4 Flag bytes
    Possible values
        0 = NO FLAGS
        TODO: add flags
1 Section size byte
2 Section number bytes
2 Name section bytes
Section header total 40 bytes
1 Type byte
    Possible values
        0 = INVALID
        1 = NAME SECTION
        2 = STRING SECTION
8 Virtual addr bytes
8 Memory size bytes
8 File size bytes
8 String table index bytes
4 Offset in file size bytes
1 Flags byte
    Possible values
        0 = NONE
        1 = LOAD
        2 = EXECUTE
        4 = WRITE
2 Allignment bytes
Symbol Entry 26 bytes total
8 String table name index bytes
8 Value bytes
8 Size bytes
1 Info byte
1 Other info byte
